# Security Policy

## Supported Versions
This repository documents an experimental contract series (v0–v10) deployed on Base.
Artifacts are provided for educational and demonstration purposes.
No active maintenance or patch guarantees are provided.

## Reporting a Vulnerability
If you believe you’ve found a vulnerability, please open a private report:
- Email: security@yourdomain.example (or GH Security Advisory)
- Include version (vX), network, contract address, and steps to reproduce.

We will acknowledge receipt within 5 business days and follow up on the status.
Please avoid public disclosure until we coordinate a fix or mitigation.
